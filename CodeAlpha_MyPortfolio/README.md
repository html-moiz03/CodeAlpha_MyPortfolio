# 🌐 Malik Moiz — Frontend Developer Portfolio

A personal portfolio website built with pure **HTML**, **CSS**, and **Vanilla JavaScript** — created as Task 3 of the **CodeAlpha Front-End Development Internship**.

---

## 🔗 Live Demo

[View Live](https://malik-abdulmoiz-portfolio.netlify.app)

---

## ✨ Features

- ⌨️ Typing animation in hero section
- 🎨 Clean dark theme with cyan accent color
- 📱 Fully responsive — mobile, tablet, desktop
- 🍔 Hamburger menu for mobile navigation
- 🔗 Sticky navbar with scroll-based frosted glass effect
- 🔆 Active nav link highlights on scroll
- 🎞️ Scroll-reveal animations on all cards
- 📊 Animated skill bars on scroll
- 📋 Projects section with GitHub and live demo links
- 📬 Contact form with success message
- ♿ Accessible — ARIA labels, focus styles, reduced motion support

## 🛠️ Built With

- HTML5 (semantic markup, ARIA accessibility)
- CSS3 (custom properties, flexbox, grid, animations)
- Vanilla JavaScript (IntersectionObserver, typing effect, scroll events)

---

## 🎨 Design

- **Color scheme:** Cyan & Black (`#00d4ff` on `#0c0c0f`)
- **Fonts:** Space Grotesk (headings) + Inter (body) + JetBrains Mono (code/labels)
- **Signature element:** Character-by-character typing animation in the hero
